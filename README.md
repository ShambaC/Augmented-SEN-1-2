## Activate Python environment
```bash
# Clone repository
git clone https://github.com/ShambaC/SAR-Image-Colorisation.git

# Change to the python environment directory
cd PythonENV

# Create environment (one time)
python -m venv .

# Activate environment (OS based)
cd Scripts

# Windows
activate.bat

# Linux
chmod +X activate
./activate
```
